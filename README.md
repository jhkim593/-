# led-matrix 방향지시등
플렉스 센서 구부러짐이 구부러짐이 설정 범위를 벗아나면 led-matrix와 lcd가 출력되는 장치이다.<br/>
플렉스 센서는 2개를 이용하였으며, 택트스위치를 이용하여 스위치가 HIGH인 상태에서만 입력센서가 <br/>
작동하도록 하였다.
